About

I'm a UX & UI designer based in Manchester. For the past 3 years I've helped individuals, startups and companies reach their goals with user focused design. I specialize in designing web & mobile interfaces that look great and are easy to use. While I enjoy design, I also have a passion for bringing experiences to life. To that end, I've developed my front end development skills thanks to courses and communities like FreeCodeCamp, Udemy, and many others. My goal is to join an innovative UX team in an environment where I can contribute, while continuing to learn & grow.

Wikipedia Viewer
This simple web-based application fetches exerpts from Wikipedia articles based on a user-entered search term. Built using jQuery, this app makes AJAX requests using the Wikipedia API. The UI utilizes CSS animations to emphasize the search bar, and material design shadows and depth when displaying the results.
SquareTrade Retention Page
I was responsible for the UI design and coding of a responsive customer retention page for SquareTrade’s Danish and Austrian micro sites. The page underwent A/B testing alongside other retention page designs with the goal of improve customer retention through messaging & imagery. The link takes you to a sample since the real page is only accessible to SquareTrade customers.
CodePlayer
Test your JavaScript, CSS, and HTML in real time with this code editor. Similar to JSFiddle, this simple app renders the code you've entered and outputs it with the help of an iFrame. The different panels can be toggled on and off to simplify things if you're only working in one or two languages. This was a fun project that helped me to develop my jQuery skills even further.
Primer
This simple web-based application takes a user-entered number and determines if it’s a prime number or not. The main challenge of this exercise was creating an efficient mathematical algorithm for checking if a number is prime. I did this using PHP. I always enjoy styling my projects with CSS - this UI features a simple interface with bold colours.
Local Weather App
This is a simple app that shows the weather in the users current location, in units of Celcius or Farenheit. The app uses the HTML5 Geolocation API to get the user's current location. An AJAX request is sent through the Google Maps Geocoding API to get the city name that corresponds to the users coordinates. Another AJAX request gets the temperature using the freeCodeCamp Weather API.
Postal Code Lookup
This is a Postal Code Lookup app that uses the Google Maps API to obtain the postalal code that corresponds to the address entered by the user. The UI features material design inspired labels & input fields. The highlights are achieved using CSS animations and the labels are repositioned using transitions so that they're always visible to the user.
La Chaine du Droit
La Chaîne du Droit challenges the old way of tackling legal issues. It believes that legal information should be free and accessible to the public. To help them achieve this goal, I created a website prototype that is pleasant and intuitive to use.
BagBirdy
I was hired to design a website for a company in the fashion industry. BagBirdy wanted to present itself as the authority on all things handbag. I worked with BagBirdy to come up with a responsive design that would appeal to their customers.
Skillgigs
I was hired by Digital Astronauts to design a new homepage for the Skillgigs website. The goal was to appeal to both employers and prospective employees. I worked within the the companies existing website & brand identity to give their homepage a fresh new look.
FreshHost
FreshHost is a new Google Home service that helps Airbnb hosts create conversational house manuals for their guests. I worked with the company to design their logo and helped improve the design of their homepage and web-based platform.
MyCompanyWeb
From wireframes to the final mockup, I created the visual design for My Company Web’s new website. A colourful site featuring flat design. The page showcases the company’s services and entices the user to keep scrolling.
Drive Safe
From wireframes to the final mockup, I created the visual design for Drive Safe’s new mobile website. The minimal colour palette is inspired by the company’s current website. The crisp and clean design contains elements of Material Design: tangible layers created using shadows, bright accent colours, and floating icons and cards.
Host With Love
From wireframes to the final mockup, I created the visual design for Host With Love’s new website. The inviting design encourages users to sign up for the company’s services. Bright colours, flat icons, and hand-drawn patterns keep this design modern and hip. The use of white space allows the company to convey a lot of information about their services without feeling cluttered.
Zyp.li
I created the visual design for Zyp.li’s new website using Photoshop. Zyp.li wanted a clever hero image to hint at their URL shortening service. The playful theme is carried throughout the site using photos, hand-drawn elements and a friendly testimonials section. The grey & white colours give the site a modern feel, while the red accents impart a sense of urgency, calling the user to take action.
